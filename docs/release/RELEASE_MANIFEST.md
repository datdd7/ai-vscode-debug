## 5. API Surface

**Endpoint:** `POST http://localhost:9999/api/debug`

**38 Supported Operations:**

Session: `launch`, `restart`, `quit`
Execution: `continue`, `next`, `step_in`, `step_out`, `jump`, `until`
Breakpoints: `set_breakpoint`, `set_breakpoints`, `set_temp_breakpoint`,
  `remove_breakpoint`, `remove_all_breakpoints_in_file`,
  `disable_breakpoint`, `enable_breakpoint`, `ignore_breakpoint`,
  `set_breakpoint_condition`, `get_active_breakpoints`,
  `get_data_breakpoint_info`, `set_data_breakpoint`, `watch`
Threads: `list_threads`, `switch_thread`
Hardware: `get_registers`, `read_memory`, `disassemble`
Stack: `stack_trace`, `list_source`, `up`, `down`, `goto_frame`, `get_source`
Variables: `get_stack_frame_variables`, `get_args`, `evaluate`, `pretty_print`,
  `whatis`, `execute_statement`
Status: `get_last_stop_info`
